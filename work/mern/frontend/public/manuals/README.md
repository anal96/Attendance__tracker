# User Manual PDFs

This directory contains user manual PDFs for each dashboard type:

- `admin-user-manual.pdf` - Admin Dashboard User Manual
- `manager-user-manual.pdf` - Manager Dashboard User Manual  
- `employee-user-manual.pdf` - Employee Dashboard User Manual

## Note for Developers

These PDF files should be created and placed in this directory. Each manual should contain:

1. **Complete dashboard overview** - Navigation and layout
2. **All features explained** - Step-by-step guides for every function
3. **Screenshots and illustrations** - Visual guides for each feature
4. **Tips and best practices** - How to use features effectively
5. **Troubleshooting** - Common issues and solutions
6. **Keyboard shortcuts** - Quick actions and shortcuts

The PDFs are automatically served from `/manuals/[manual-name].pdf` and can be downloaded from the Settings page.
